# quant_project

A quantitative finance project for data analysis, algorithmic strategy development, and backtesting.

## Project Structure

- `data/`: Raw and processed financial datasets.
- `strategies/`: Implementation of trading algorithms and signals.
- `backtesting/`: Framework for evaluating strategy performance.
- `notebooks/`: Exploratory data analysis and research.
- `utils/`: Helper functions for data fetching and technical indicators.

## Getting Started

1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Configuration**:
   Set up your API keys (e.g., Alpaca, Yahoo Finance, or Interactive Brokers) in a `.env` file.

3. **Run Backtest**:
   ```bash
   python main.py
   ```
